Pages are used to display video content in a Brightcove Beacon application. To create a page, follow these steps:

  1. Open the Pages page.
  2. In the side navigation panel, click Create Page. The new page panel will open.
  3. Enter a Page Name using the following guidelines. Press enter to save the name.
    • Name is limited to 100 characters
    • Illegal character: /
  4. Select a Page Type by clicking on it. The Page Type controls the layout of the page and how the content is displayed on the page. Selecting a type will update the page preview on the right.
  5. Click Next.
  6. Click Add Playlist. A list of playlists that have been created will display.
  7. Select one or more playlists to add to the page by hovering over the playlist and then clicking Add. The number of playlists that can be added is based upon the Page Type that was selected. Some Page Types only allow one playlist to be added. As playlists are added, they will appear on the page preview.
  8. Click X to close the list of playlists.
  9. Click Back to All Pages to return to the list of pages.

The page that was created will display in the list of pages.

To edit any of the page details, click on the page name.
