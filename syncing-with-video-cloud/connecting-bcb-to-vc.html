Overview

Your video and playlist assets are pulled ingested from Brightcove Video Cloud. You use the Tools (wrench) menu item to do this, as shown here:

This assumes you are connected to the correct Video Cloud account. In this document you will see how to connect your Brightcove Beacon Experience to your Video Cloud account.

Obtain Video Cloud authentication credentials

To associate a Video Cloud account with Brightcove Beacon, you will need API authentication credentials. In this section you will obtain the needed credentials.

  1. Open Video Cloud Studio, which can be found by browsing to https://studio.brightcove.com/.
    Skeleton steps will be provided here to obtain your client ID and client secret. For full details, including a video on the process, see the Managing API Authentication Credentials document.
  2. From the HOME module, copy the account number. which appears just under the account name, from the top left of the page. Save the number for later use.
  3. From the ADMIN module, select API Authentication.
  4. In the API Authentication page click Register New Application.
  5. Supply a Name and Description.
  6. Check the checkbox in front of the Video Cloud account you will be associating with Brightcove Beacon.
  7. For the Exposed Brightcove APIs, click the checkboxes in front of Playlist Read and Video Read.

Connect Brightcove Beacon to Video Cloud

  1. Open Brightcove Beacon, which can be found by browning to https://beacon.brightcove.com/your_experience_name/home.
  2. In the top menu, click the (Settings) icon.
  3. In the left menu, click the Auth OVP option.
  4. Note that you need to get the following information from Video Cloud:
    • Client_Id
    • Client_Secret
    • Account_Id
