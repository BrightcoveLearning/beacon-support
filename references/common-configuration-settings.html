Common Configuration Settings

In this topic, you will learn about the common configuration settings used when creating or modifying your app in Brightcove Beacon.

Overview

This topic is a compilation of the common configuration settings associated with your Brightcove Beacon app. You will also find these settings in other documentation related to completing specific tasks in Brightcove Beacon.

Rights & Scheduling

The Rights & Scheduling section lets you define playback permissions and set publishing schedules for your app.

Here is an example of the Rights & Scheduling tab for adding or editing Pages:

This table is a compilation of the common configuration fields:

Field Required Description
Status Yes Select one of the following types:
  • Published
  • Unpublished
Rights Start date Yes Set the starting date and time for your app
Rights End Date Yes Set the ending date and time for your app
Devices to publish Yes Add the devices you wish to publish to. Select the Select / Unselect All Devices to see a list of available devices.
Permitted locations Yes Select the geographic locations where you want to allow users to view your content. Click inside the Permitted locations box to see a list of locations.
Denied locations No Select the geographic locations where you DO NOT want to allow users to view your content. Click inside the Denied locations box to see a list of locations.

To add or edit geographic locations, navigate to Settings and then Locations.

Textual Data

The Textual Data section allows you to add displayable text for your page.

  1. Select the Textual Data tab.

  2. Add values for the following fields:

    Field Required Notes
    English Title Yes The displayed title for your page

Non-Textual Data

The Non-Textual Data section allows you to upload icon images for your page.

  1. Select the Non-Textual Data tab.

  2. Choose images from your computer to be used as page icon images.

    • Icon
    • Hover Icon
    • Horizontal Background
    • Vertical Background

    The maximum allowed upload size is 500 KB.
