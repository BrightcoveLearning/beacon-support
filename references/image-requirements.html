Video Cloud images

Whenever a video is ingested into Video Cloud two images are automatically created and shown in the video's properties. The two images created are a Poster and a Thumbnail. This is the IMAGES section in Studio in edit mode.

The Poster and a Thumbnail are placeholders for the images used in Brightcove Beacon. DO NOT follow the recommendations for the images as shown in Video Cloud Studio! Instead use these guidelines:

Poster Thumbnail
Landscape image (16:9):
Minimum Width: 1000px Height: 563px
Maximum Width: 4000px Height: 2250px
Poster image (2:3)
Minimum Width: 1000px Height: 563px
Maximum Width: 4000px Height: 2250px
To summarize, you need to replace the images automatically created by Video Cloud Studio to images matching the requirements shown in the table.

Corporate logo

You will need to supply your corporate logo for use in the Brightcove Beacon Experience. Follow these guidelines for the image:

Specification Value
Format .png
Aspect Ratio 1:1
Size 1500px x 1500px

Working in Brightcove Beacon

When entering information in Brightcove Beacon you will be asked for Non-Textual Data. This means images for the assets on which you are currently creating. For instance, this will happen when creating the following:

  • Series
  • Season
  • Movie
  • Live Event
