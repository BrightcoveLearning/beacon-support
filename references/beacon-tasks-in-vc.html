Task summary

The following is the list of tasks that you as Brightcove Beacon customers should/must do in Video Cloud

  • Add video assets
  • Create custom fields
  • Add video metadata
  • Add images associated with the video asset
  • Add rights management information
  • Add episode data for a series/season to a video asset
  • Create video playlists
  • Create Brightcove Live events
  • Create an SSAI ad configuration
  • Implement Cloud Playout

Each one of these will be further explained later in this doc with a link to the documentation where instructions live on how to perform that particular task.

This doc is NOT a "how to" manual for each task, but each section will link you out to the appropriate doc that contains steps on how to complete the task.

Add video assets

Obviously your Brightcove Beacon app would not be very interesting with video content, so this is a hugely important task. Video Cloud is the tool used to upload your videos. Later in this doc you will see Video Cloud Studio is a commonly used tool when working with your Brightcove Beacon app.Remember that Video Cloud is the "truth" of Brightcove Beacon video content.
