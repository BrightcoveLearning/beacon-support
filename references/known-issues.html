Known Issues

This topic lists known issues for the {{ site.product }} application.

Parental controls

  1. All content is displayed to end users (in lists, carousels, etc), but when trying to play the video the parental pin will be requested.

Mobile web

  1. Mobile web has limited support. On devices with a width under 375px a message is displayed to install the appropriate native Brightcove Beacon app.
