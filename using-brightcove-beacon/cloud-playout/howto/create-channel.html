Add (to create) or edit the settings for the channel:

Name

Enter a name (English only, no spaces; the following special characters are allowed: _ . : / = + -). Required

Description

Enter a description of the channel.

Image URL

Enter the URL for an image to represent the channel. If you do not, a default image will be used.

Schedule

Click on the Start Date/Time field to select a start date and time. Required

If you toggle Loop Playout Content on, you may also enter an optional End Date/Time

Ad Insertion

Toggling this on enables SSAI - it will also remove the Live Input option.
