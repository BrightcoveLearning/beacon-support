Add new channel
  1. Click to open the Create Channel dialog.
  2. Enter a name (English only, no spaces; these special characters are allowed: _ . : / = + -).
  3. Enter a description of the channel.
  4. Click on the Start Date/Time field to select a start date and time.
  5. If you toggle on, you may also enter an optional End Date/Time
  6. Toggle on to enable SSAI - it will also remove the Live Input option.
  7. Toggle to include live events in your channel. You will also need to provide a URL for slate content or use the default slate to fill time when the live stream isn't available.
  8. Select Mobile & Web and/or Smart TV for the output.
  9. Select the Output Region closest to the majority of your audience.
  10. Choose whether to output to:
    • Brightcove Live (select an existing event or generate a new one)
    • RTMP (enter the RTMP URL for YouTube, Facebook or another service)
    • An S3 bucket (enter the public/private access keys and region)
  11. Click to create the channel.
