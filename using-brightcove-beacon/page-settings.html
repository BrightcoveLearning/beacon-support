Common Page Configuration Settings

In this topic, you will learn about the common settings when creating or modifying a Brightcove Beacon Page.

Overview

The Brightcove Beacon page is where users select and watch your video content. This topic contains the common page settings found under the following tabs:

  • Textual Data
  • Non-Textual Data
  • Tags
  • Rights & Scheduling

Textual Data

The Textual Data section allows you to add displayable text for your page.

  1. Select the Textual Data tab.

  2. Add values for the following fields:

    Field Required Notes
    English Title Yes The displayed title for your page

Non-Textual Data

The Non-Textual Data section allows you to upload icon images for your page.

  1. Select the Non-Textual Data tab.

  2. Choose images from your computer to be used as page icon images.

    • Icon
    • Hover Icon
    • Horizontal Background
    • Vertical Background

    The maximum allowed upload size is 500 KB.

Tags

This section, focused on SEO, is not functional at this time.

Rights & Scheduling

The Rights & Scheduling section is where you set the publish status and select scheduling, devices and locations.

  1. Select the Rights & Scheduling tab.

  2. Add values for the following fields:

    Field Required Notes
    Status Yes Select one of the following types:
    • Published
    • Unpublished
    Rights Start date Yes Schedule a starting time for your page
    Rights End Date Yes Schedule an end time for your page
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.

    To add or edit geographic locations, navigate to Settings and then Locations .
