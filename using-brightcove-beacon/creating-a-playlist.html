Overview

Playlists are a collection of videos that are created one of two ways in Brightcove Beacon:

  • Imported into Brightcove Beacon from your Video Cloud library.
  • Created in Brightcove Beacon using the Playlists tab. This method is the focus of this document.
Not all the functionality shown in the Playlists tab is usable at this time. It is made clear in this document what should NOT be used.

Basic Data section

To create a playlist in Brightcove Beacon, follow these steps:

  1. Login to Brightcove Beacon.
  2. In the top navigation, select Playlists.

  3. Select the Add new playlist button.

  4. From the Basic Data tab, you will define the name and display type of playlist, then add content. Note that most of the functionality in this section is not usable at this time. Of course, the steps below will only address the usable items.

  5. In the Name field, add an internal system name for your playlist. This is also the name use for analytics. Note that you will add the viewable title in a later step.
    It is a recommended best practice to somehow designate the playlists created in Brightcove Beacon vs. Video Cloud. For instance, simply place an asterisk (*) in front of playlist name when the playlist is created in Brightcove Beacon. Of course, it does not have to be an asterisk, but choose some notation so that when it comes time to update a playlist you know to go to Brightcove Beacon Playlists tab or Video Cloud Studio to make alterations.
  6. Select a layout from View All Playlist Layout. This is the layout used when viewers click the View All button in the apps. As a best practice choose a grid of some kind.
  7. To begin the process of adding content to your playlist, click the Assets tab. You see you have six asset types to choose from.
    You are now at the point where you see playlist options, those being Events, Series, Seasons, Events and Channels, that you cannot choose in Video Cloud Studio. These types are NOT accessible in Video Cloud Studio, so you must create the playlist in Brightcove Beacon.

  9. Be sure to click the Create New Playlist button to save your work.

Textual Data

The Textual Data section allows you to add displayable text for your playlist.

  1. Select the Textual Data tab.

  2. Add values for the following fields:

    Field Required Notes
    Title Yes The displayed title for your playlist.
    Description Yes The displayed description for your playlist.
    Subtitle No A short synopsis about your playlist. This has a maximum of 188 characters.

Non-Textual Data

The Non-Textual Data tab is not functional at this time.

Tags

The Tags tab is not functional at this time.

Rights & Scheduling

The Rights & Scheduling section is where you set the publish status and select the payment type for your playlist.

  1. Select the Rights & Scheduling tab.

  2. Be sure the status is set to Published for your playlists.

  3. In the Availability Rights section, select a payment method:

    • No Payment
    • Stripe
    • Roku Store
    • Apple Store
    • Google Play Store
    • Amazon

  4. When No Payment is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.

    To add or edit geographic locations, navigate to Settings and then Locations.

  5. When a payment option is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.
    Price Yes Enter the currency amount.
    Currency Yes The currency type. Use the Add Currency button to add other currencies.
    Vendor ID Yes Enter your vendor ID. In this example, it is your Amazon ID.

    To add or edit geographic locations, navigate to Settings and then Locations.

Advertisement

The Advertisement section allows you to link an ad configuration to your playlist.

  1. Select the Advertisement tab.

  2. Choose whether you want ads associated with your playlist:

    • Yes - Select an ad configuration from the Advertisement Configuration dropdown menu.
    • No - No ads.

    The advertisement configurations displayed in the dropdown menu are created in the Advertisement Configuration section. For details, see the Brightcove Beacon Advertisement Configuration document.
