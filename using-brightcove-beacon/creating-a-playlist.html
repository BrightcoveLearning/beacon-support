Overview

Playlists are a collection of videos that are created one of two ways in Brightcove Beacon:

  • Imported into Brightcove Beacon from your Video Cloud library.
  • Created in Brightcove Beacon using the Playlists tab. This method is the focus of this document.
Not all the functionality shown in the Playlists tab is usable at this time. It is made clear in this document what should NOT be used.

Basic Data section

To create a playlist in Brightcove Beacon, follow these steps:

  1. Login to Brightcove Beacon.
  2. In the top navigation, select Playlists.

  3. Select the Add new playlist button.

  4. From the Basic Data tab, you will define the name and display type of playlist, then add content. Note that most of the functionality in this section is not usable at this time.

  5. In the Name field, add an internal system name for your playlist. This is also the name use for analytics. Note that you will add the viewable title in a later step.
    It is a recommended best practice to somehow designate the playlists created in Brightcove Beacon vs. Video Cloud. For instance, simply place an asterisk (*) in front of playlist name when the playlist is created in Brightcove Beacon. Of course, it does not have to be an asterisk, but choose some notation so that when it comes time to update a playlist you know to go to Brightcove Beacon Playlists tab or Video Cloud Studio to make alterations.

  6. Select a layout from View All Playlist Layout. This is the layout used when viewers click the View All button in the apps. As a best practice choose a grid of some kind.

  7. Select a View all menu type. If the playlist is long, and only a few videos are displayed at a time, you can give users the ability to view all based on genres. Options include:

    • None - No option to view all videos
    • Tab - Tabbed list of all videos
    • Combobox - A dropdown menu where you can view the titles of all the videos

  8. Select a Playlist Reference.

    • No Reference - default
    • Child Navigation - default
    • Movies Sub Page - default
    This is used to allow multilevel navigation. In the apps, when users press View All on the playlists that have a Playlist Reference, instead of the viewing the normal page to view all, users are taken to the selected type from this selection.
  9. Neither Dynamic nor Special playlists are functional at this time, so do not check either checkbox.

Use Builder to add content

The Builder tab and functionality allows you to add content to your playlist based on a set of rules that you define.

  1. By selecting Special playlist you will have to option to choose a special type of playlist.

  2. Select a Special Type of playlist.

    • Most Watched - Most watched videos playlist
    • Recommendations - Recommended videos playlist
    • Builder - Use the Builder tab below to create your playlist content
  3. If you choose Builder as special type, then you can use the Builder functionality to add content to your playlist.

  4. Select the builder options and select the Add button. Then, select the Update List button to see content added to your playlist on the left.

Manually add content

When the Special playlist option is not selected, you can use the other tabs to add content to your playlist.

  1. Make sure Special playlist is not selected.

  2. The Assets tab allows you to add content from your library of movies, series, seasons, episodes, channels and advertisements.

  3. In the Actions column, use the add icon to add an item to your playlist.

    Use the remove icon to remove an item from your playlist.

  4. The Seo-Tags tab allows you to choose content that has tag information associated with it.

  5. The Cast tab allows you to choose content based on actors, directors, music composers, singers song writers and writers.

  6. The Categories tab allows you to choose content based on categories that you have defined.

  7. The Genres tab allows you to choose content based on genres and sub-genres.

  8. Select Newest first to order your playlist by asset creation date.

Textual Data

The Textual Data section allows you to add displayable text for your playlist.

  1. Select the Textual Data tab.

  2. Add values for the following fields:

    Field Required Notes
    Title Yes The displayed title for your playlist.
    Description Yes The displayed Description for your playlist.
    Subtitle No A short synopsis about your playlist. This has a maximum of 188 characters.

Non-Textual Data

The Non-Textual Data tab is not functional at this time.

SEO Data

The SEO Data section is where you add information related to search engine optimization (SEO).

  1. Select the SEO Data tab.

  2. Add values for the following fields:

    Field Required Notes
    Friendly URL Yes This is generated automatically from the title you entered in the Textual Data section.
    SEO Title No The title used by search engines.
    SEO Keywords No Enter one or more keywords to make your content more easily found by search engines.
    SEO Description No The description used by search engines.

Rights & Scheduling

The Rights & Scheduling section is where you set the publish status and select the payment type for your playlist.

  1. Select the Rights & Scheduling tab.

  2. Select a status:

    • Published - Your playlist is published to your apps.
    • Unpublished - Your playlist is not published.
    • Pre-Prod - Your playlist is set to pre-production status.

  3. In the Availability Rights section, select a payment method:

    • No Payment
    • Stripe
    • Roku Store
    • Apple Store
    • Google Play Store
    • Amazon

  4. When No Payment is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.

    To add or edit geographic locations, navigate to Settings and then Locations.

  5. When a payment option is selected, enter values for the following fields:

    Field Required Notes
    Rights Type Yes Select one of the following types:
    • Free - Free content
    • Advertising - AVOD
    • Subscription - SVOD
    Start Time Yes Schedule a starting time for your playlist.
    End Time Yes Schedule an end time for your playlist.
    Max Streams Yes The maximum number of streams per user.
    Devices to publish Yes Add the devices you wish to publish to. Select the Select/ Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content.
    Price Yes Enter the currency amount.
    Currency Yes The currency type. Use the Add Currency button to add other currencies.
    Vendor ID Yes Enter your vendor ID. In this example, it is your Amazon ID.

    To add or edit geographic locations, navigate to Settings and then Locations.

Advertisement

The Advertisement section allows you to link an ad configuration to your playlist.

  1. Select the Advertisement tab.

  2. Choose whether you want ads associated with your playlist:

    • Yes - Select an ad configuration from the Advertisement Configuration dropdown menu.
    • No - No ads.

    The advertisement configurations displayed in the dropdown menu are created in the Advertisement Configuration section. For details, see the Brightcove Beacon Advertisement Configuration document.
