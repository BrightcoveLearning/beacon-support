Steps to Create a new Coupon

To create new coupons/vouchers in Brightcove Beacon, please follow the next steps:

  1. Go to your Brightcove Beacon account and click the Commerce tab.
  2. In the left menu, select Coupons by Stripe.
  3. Click the Add new Voucher button.
  4. In the Id field, type the code that users will use.
  5. For the Name field use a User-friendly name that can be displayed in invoices.
  6. The Duration field represents the value duration of the coupon in months.
  7. For the Discount type, you have the option to specify the discount in currency or percentage. For example, If the discount is 3 months free, then you set the duration to Repeating, with 3 months and discount type as 100%.
  8. In the Limitations section, you can limit the number of redemptions or date to use the coupon. For example, you can say that the coupon can only be used until the end of August, or that only the first 100 people will get the coupon.
  9. In the Packages/Subscription Associated section, you can specify the subscriptions that can be used with the coupon. For example, you can say that the coupon is only valid if you buy the Premium Yearly subscription. If you do not select a subscription then it will apply to all package subscriptions.
  10. Once the Coupon/Voucher has been created, any registered user can apply it when buying a subscription.
