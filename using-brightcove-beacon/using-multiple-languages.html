Define the languages for your apps

Defining the languages you wish to use in your Brightcove Beacon apps is part of the onboarding process. Past the onboarding process, if you choose to start using multiple languages, or wish to add languages, please contact your Brightcove support person.

App Translations needed for your experience

The following will need to be localized for the languages you are supporting:

  • Page Titles - In this example localizing for the Home page
  • Page images Not possible in Beacon Studio
  • Playlist titles
  • Channel titles Not possible in Beacon Studio
  • Channel images Not possible in Beacon Studio
  • Series titles

What is localized?

There are two parts of the apps that are/can be localized, they are:

  • The app UI itself, which is localized for you
  • Items you create, like page names and video metadata, for which you must supply the translated terms

How your viewers choose a language for the app

There are two ways your viewers can choose the language to use in the app. They are:

  • Select the language in the Settings page
  • Set the language of the browser

Change the language in the app

Viewers can choose the language in the Brightcove Beacon app itself. The following shows the process the viewer would follow using the Settings page to change the app language from English to German:

Set the preferred language in the browser

If a viewer sets a preferred language in a browser, and that language is supported in the Brightcove Beacon app, the app will automatically use that language. The viewer would not have to set the language in the Settings standard page.
