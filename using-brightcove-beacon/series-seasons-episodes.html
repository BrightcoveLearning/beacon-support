Overview

When creating a series you must define the relationships between the series and its seasons, and the episodes in each season. A high-level view of the process is:

  1. In Brightcove Beacon, create the series. This involves not only naming the series, but also providing information such as images to be used for the series in the app, dates the series can be viewed, etc.
  2. In Brightcove Beacon, create the seasons related to the series. You will provide information similar to the series, with the addition of selecting the series to which the season belongs.
  3. In Video Cloud Studio, you will assign the video episodes to the corresponding season via values supplied to custom fields. You will then ingest those videos into the Brightcove Beacon experience. The fields to be assigned data are:
    • beacon.episode.number
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

Creating a series

To create a series, follow these steps:

  1. Log in to Brightcove Beacon and select Programs.
  2. In the left navigation, select Series.
  3. Click the Add new series button.

  4. On the Basic Data tab, fill in the following:

    Form Field Name Description / Requirements
    Name The title of the series
    Production Year The year the series was produced; this information will appear in the the More Info screen
    Production Country The country where the series is produced; this information will appear in the the More Info screen
    Genres The genres of the series; note that you can create genres as needed:
    Sub-Genres Do not complete, this data is not used
    Genre Language Preview The language of the preview
    Episode Sort type How to sort episodes in the app user interface
    Cast The metadata under CAST will appear in the the More Info screen.
  5. Be sure to click the Create New Series button to save the data entered on this form.
  6. Click on the Textual Data tab.
  7. For each language your apps are supporting, fill in the Title, Description and Synopsis. This data will be displayed on your series information page.
  8. Be sure to click the Create New Series button to save the data entered on this form.
  9. Click on the Non-Textual Data tab.
  10. Supply an image for the Portrait Poster and Landscape Poster matching the requirements below. You DO NOT need to supply images for Thumbnail and Wide banner.
    Image Requirements
    Portrait Poster Poster image (2:3)
    Minimum Width: 1000px Height: 563px
    Maximum Width: 4000px Height: 2250px
    Landscape Poster Landscape image (16:9):
    Minimum Width: 1000px Height: 563px
    Maximum Width: 4000px Height: 2250px
    Note that you can have different images for English, French and German.
  11. The information on the Tags tab is not used at this time, so do not enter information on this tab.
  12. The information on the Metadata tab is not used at this time, so do not enter information on this tab.
  13. Click on the Streams tab, where you will select a trailer for the series. The following screenshot will help guide you through the process of selecting that trailer.
  14. Select your Provider. You will most likely only have one provider, which references your Brightcove Video Cloud account.
  15. For the Stream Video ID, obtain the video ID for the trailer from Studio and place it in the text box.
  16. Be sure the Stream Type is set to Trailer.
  17. Note that the DRM Type and Stream Format values do not need to be altered.
  18. Be sure to click the Create New Series button to save the data entered on this form.
  19. Click on the Rights & Scheduling tab, as shown here.

  20. Add values for the following fields:

    Field Required Notes
    Status Yes Select one of the following types:
    • Published
    • Unpublished
    Rights Type Yes Choose the appropriate value for your series, from the options:
    • Advertising - AVOD
    • Free
    • Subscription - SVOD
    Start Time Yes Schedule a starting date your series will be available
    End Time Yes Schedule an ending date when your series will no longer be available
    Max Streams Yes The maximum number of streams that can be watching the series from the same account
    Devices to publish Yes Add the devices you wish to publish to. Select the Select / Unselect All Devices to see a list of available devices.
    Permitted locations Yes Select the locations where you want to stream your content.
    Denied locations No Select the locations where you DO NOT want to stream your content. Enter World if you have no restrictions.

    To add or edit geographic locations, navigate to Settings and then Locations.
  21. Be sure to click the Create New Series button to save the data entered on this form.

You have now created your series. The next step is to create one or more seasons that belong to this series. The next section leads you through that process.

Creating seasons

To create a season, follow these steps:

  1. Log in to Brightcove Beacon and select Programs.
  2. In the left navigation, select Seasons.
  3. Click the Add new season button.
  4. On the Basic Data tab fill in the following:
    Form Field Name Description / Requirements
    Select a Series The dropdown will be populated with the series you have created. Choose the series to which this season belongs.
    Name Enter a name for the season. You can enter a name like Season 1, but if a series has topics for distinct seasons that may be a better choice. For instance, a series on the continents where each season focuses on a continent, good names for the seasons would be Europe, Africa, Asia, etc.
    Season Number This must be an integer number representing the season sequence in the series. This number will be used when ingesting episode videos from Video Cloud.
    Production Year The year the series was produced; this information will appear in the the More Info screen
    Production Country The country where the series is produced; this information will appear in the the More Info screen
  5. Be sure to click the Create New Season button to save the data entered on this form.
  6. Click on the Textual Data tab.
  7. For each language your apps are supporting, fill in the Title, Description and Synopsis. This data will be displayed on your season information page.
  8. Be sure to click the Create New Season button to save the data entered on this form.
  9. Click on the Non-Textual Data tab.
  10. Supply an image for the Portrait Poster and Landscape Poster matching the requirements below. You DO NOT need to supply images for Thumbnail and Wide banner.
    Image Requirements
    Portrait Poster Poster image (2:3)
    Minimum Width: 1000px Height: 563px
    Maximum Width: 4000px Height: 2250px
    Landscape Poster Landscape image (16:9):
    Minimum Width: 1000px Height: 563px

    Note that you can have different images for English, French and German.
  11. The information on the Tags tab is not used at this time, so do not enter information on this tab.

  17. Be sure the Stream Type is set to Trailer.

You have now created one of your seasons. If needed, create more seasons for the series. The next step is to ingest episode videos from Video Cloud into Brightcove Beacon. The next section leads you through that process.

Ingesting episodes

As you have seen, creating the series and associated seasons is done in Brightcove Beacon. Creation of episodes for the series is done in Video Cloud Studio. You will assign values about the series and seasons in custom fields for each episode video, then ingest those videos into Brightcove Beacon. If you have not created the custom fields in Video Cloud Studio, see the Creating Video Cloud Custom Fields document. To ingest episodes, follow these steps:

These steps assume you have already uploaded your episode videos to Video Cloud.
  1. Log in to Video Cloud Studio.
  2. From the dropdown menu at the top of the page, select MEDIA.
  3. Click on the title of one of the episode videos to enter the video's properties page.
  4. Scroll down until you see on the right-hand side the CUSTOM FIELDS section.
  5. Click the Edit button.
  6. Locate the custom fields that are associated with episodes.
  7. For the episode video, enter the series name, season number and episode number in the corresponding custom field.
  8. Be sure to click the Save button after entering the appropriate values for the video.
  9. Repeat entering values in the custom fields for all your episode videos.
  10. Log in to Brightcove Beacon.
  11. Click on the Tools option (wrench icon) in the top menu.
  12. Be sure the Ingestion option is selected on the left menu.
  13. Click the Update Brightcove Videos button to ingest the newly configured episode videos.

  14. To make the videos immediately available in your apps, clear the Brightcove Beacon Experience cache by clicking on the Cache option in the left navigation, then click the Cache Purge button.

    For more details on using the Brightcove Beacon Tools options, see the Using the Tools document.
