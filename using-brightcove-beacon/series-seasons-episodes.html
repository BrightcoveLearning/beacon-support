Overview

When creating a series you must define the relationships between the series and its seasons, and the episodes in each season. A high-level view of the process is:

  1. In Brightcove Beacon create the series. This involves not only naming the series, but also providing information such as images to be used for the series in the app, dates the series can be viewed, etc.
  2. In Brightcove Beacon create the seasons related to the series. You will provide information similar to the series, with the addition of selecting the series to which the season belongs.
  3. In Video Cloud Studio you will assign the actual video episodes to the corresponding season via values supplied to custom fields, then ingest those videos into the Brightcove Beacon experience. The fields to be assign data include:
    • beacon.episode.number
    • beacon.episode.seasonName
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

The rest of this document will provide details for each of these steps.

Creating a series

To create a series, follow these steps:

  1. Log in to Brightcove Beacon and select Programs.
  2. In the left navigation, select Series.
  3. Click the Add new series button.
  4. On the Basic Data tab fill in the following:
    Form Field Name Description /
    Requirements
    Name The title of the series
    Production Year The year the series was produced
    Production Country The country where the series is produced
    Genres The genres of the series; note that you can create genres as needed assets/images/using-bcb/series-seasons-episodes/make-genres.png
    Sub-Genres Do not complete, this data is not used
    Genre Language Preview The language the preview uses
