Adding a trailer

Adding a trailer for a Beacon movie is simple. Follow the steps below.

  1. Upload the trailer video to your Video Cloud account.
  2. You do not need to add special Beacon custom fields to the trailer video. All you need is the video id:
  3. For the video that is used for the Beacon movie, add the following custom field:
    Trailer Custom Field
    Display Name Internal Name Value
    beacon.trailer.id beacon_trailer_id (Video id for trailer)
