Creating a Commerce Project

In this topic, you will learn how to configure a project using the Commerce tab for Brightcove Beacon.

Overview

A project is built and then is used in Video Cloud to assign.

Steps to create a package

Step through the following steps to create a new project:

  1. In Brightcove Beacon, select the Commerce tab, be sure Packages is selected in the left navigation, then click Add New Package .
