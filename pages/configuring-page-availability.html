Configuring Page Availability

In this topic you will learn how to configure page availability.

Page availability settings can be used to control:

  • The dates a page is available
  • The device the page is available to be viewed on
  • The locations where the page can be viewed

To configure the Availability settings for a page, follow these steps:

  1. Open the Pages page.
  2. Click on the page name to edit. The page details will be displayed.
  3. Click on Availability to display the Availability settings.

After Availability settings have been configured, they will display in the left margin.

Click Clear All Restrictions to remove all Availability settings.

Configuring the page schedule

By default, when a page is created, it will always be available for viewing. Pages can be configured with a schedule that defines the start date and time that a page will be available for viewing. Optionally, an end date and time can also be configured. To configure a schedule for a page, follow these steps:

  1. In the Availability section, click on Schedule. The schedule settings will display.
  2. Click the calendar icon () and pick a start date.
  3. Click the clock icon () and pick a start time.
  4. (Optional) Check the End Time checkbox and select an end date and time.
  5. Click Save.

Configuring device availability

By default, when a page is created, it will be available for viewing on all devices. Pages can be configured so they can only be viewed on specific devices. To configure the devices a page can be viewed on, follow these steps:

  1. In the Availability section, click on Devices. The device settings will display.
  2. Select the devices that can view the page.
  3. Click Save.

Configuring location availability

By default, when a page is created, it will be available for viewing in all geographic locations. Pages can be configured so viewing is permitted or denied in specific locations. To configure the locations for a page, follow these steps:

  1. In the Availability section, click on Location. The location settings will display.
  2. In the Permitted Locations section, click + Location.
  3. Type the name of a country. As you begin typing the list will filter.
  4. Click on a country to select it.
  5. In the Denied Locations section, click + Location and add the countries where the page cannot be viewed.
  6. Click Save.

To remove a country, click the X next to the country.
