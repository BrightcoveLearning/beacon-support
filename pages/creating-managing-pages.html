Creating and Managing Pages

In this topic you will learn how to create and manage pages.

Pages are used to display video content in a Brightcove Beacon application. To create and manage pages, open the Pages page by clicking Pages in the left navigation. A list of pages that have been created will be displayed in the left margin.

Creating pages

  1. Open the Pages page.
  2. In the side navigation panel, click Create Page. The new page panel will open.
  3. Enter a Page Name using the following guidelines. Press enter to save the name.
    • Name is limited to 100 characters
    • Illegal character: /
    Note: After you enter the page name, a system generated, internal name is also created and will be displayed below the page name. This internal name cannot be changed.
    Note: If your account is configured with multiple languages, enter a title using the primary language here. Page names for any additional languages can be entered by editing the page title after the page has been created.
  4. The Page Type controls the layout of the page and how the content is displayed on the page.
    • Tabs - Playlists are organized into tabs that display your content in a multi-column grid of thumbnails
    • Grid - Playlist are displayed as a multi-column grid of thumbnails
    • Hero (Autoplay Channel) - One playlist is featured as a fullscreen autoplaying video with additional playlists displayed as carousels
    • Hero (Image) - One playlist is featured as a fullscreen image with additional playlists displayed as carousels

    Select a Page Type by clicking on it. Selecting a type will update the page preview on the right.

  5. Click Next.
  6. The page settings that can be configured will be displayed in the left margin. For more infomation on adding videos to a page and configuring page availability, see one of the following topics:
    • Adding Videos to Pages
    • Configuring Page Availability

    Click Back to All Pages to return to the list of pages.

The page that was created will display in the list of pages.

Editing pages

To edit a page, in the side navigation panel, click on the page to edit. The page settings will be displayed.

Editing the page title

To edit the page title, click the edit icon () next to the page name and enter a new name.

Editing the page type

To edit the page type, click Change next to the current page type. Click on new Page Type and then click Select.

Duplicating pages

Duplicating pages can be useful when you need to create a new page that is similar to an existing page. To duplicate a page, hover over the page name and click onÂ Duplicate.

Rearranging page order

The order of the pages in the left navigation determines the order of the display in a Beacon application. To change the page order, click on a page and drag it to a new location.

Hiding pages

Hiding a page will cause it to not display in a Beacon application. Hiding a page does not delete the page. To hide a page, follow these steps:

  1. Click on a page to open the page details.
  2. Click Hide Page at the bottom of the page details.

Pages that are hidden will display a hidden icon () next to them.

To show a hidden page, click on the page to open the page details and then click Show Page.

Archiving and deleting pages

Archiving a page will remove it from the application but it will not delete the page. To archive a page, follow these steps:

  1. Click on a page to open the page details.
  2. Click Archive Page at the bottom of the page details.

The total number of archived pages will appear at the bottom of the left margin.

Click Archived Pages to display a list of archived pages.

To permanently delete an archived page, click Delete. Click Restore to unarchive a page and add it back to the page listing.
