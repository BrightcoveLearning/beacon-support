Samsung

  • Store details: http://seller.samsungapps.com/tv/app/registration/list
  • Developer Overview: https://developer.samsung.com/tv/distribute/
  • Developer Account Setup: https://program.developer.samsung.com/sign-in/
  • Suggested lead time for approval: Three weeks

Required information for Samsung:

  • Applications
    • Tizen – Widget file
  • Description
  • Icons for 16 Model Group
  • Logo Asset
    • With Transparency
    • 1920x1080 pixel 32 bit
    • PNG (RGBA) with 300kb
  • Background Image
    • 1920x1080 pixel 24 bit
    • PNG (RGBA) with 300kb
  • Icon For Legacy Model Group
  • Icon
    • 512x423 pixel
    • 300kb
  • Screenshot images (x4)
  • 1280x720 or 1920x1080 pixel, 500kb
  • Description
  • PowerPoint submission document – Needs to be uploaded on the portal when submitting
