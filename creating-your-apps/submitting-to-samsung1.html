Introduction

With Brightcove Beacon, you will receive a generated device app which can be added to the Samsung App Store.

  • Store details: http://seller.samsungapps.com/tv/app/registration/list
  • Developer Overview: https://developer.samsung.com/galaxy-store
  • Developer Account Setup: https://program.developer.samsung.com/sign-in/
  • Suggested lead time for approval: Three weeks

Required information for Samsung:

  • Applications
    • Tizen – Widget file
  • Description
  • Icons for 16 Model Group
  • Logo Asset
    • With Transparency
    • 1920x1080 pixel 32 bit
    • PNG (RGBA) with 300kb
  • Background Image
    • 1920x1080 pixel 24 bit
    • PNG (RGBA) with 300kb
  • Icon For Legacy Model Group
  • Icon
    • 512x423 pixel
    • 300kb
  • Screenshot images (x4)
  • 1280x720 or 1920x1080 pixel, 500kb
  • Description
  • PowerPoint submission document – Needs to be uploaded on the portal when submitting

Getting started

To access the Samsung developer pages, follow these steps:

  1. In a browser, navigate to the Samsung Sign in page.

  2. Either sign in with your account or create a new account.

  3. When you register, you need to request a Partner account with Samsung to be able to submit your apps worldwide. Otherwise, you will be limited to a certain location.

Registering your app

Register and submit your app with Samsung.

  1. From the home page, select the Web App Registration button.

  2. You will see a set of steps to complete before you can submit your app.

  3. Select Public when you want the channel to go to the store. Select Non-certified when you want to create a testing channel.

    The channel name must be unique across both Public and Non-certified channels.

    We recommend the following:

    • Create your Public channel name, for example Destinations.

    • Create a Non-certified testing channel with a suffix of QA. So, in this case it would be named Destinations QA.

    First, we are creating the public channel.

  4. Next, we are creating the testing channel.
