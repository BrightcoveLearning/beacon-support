Introduction

The Brightcove Beacon page is where users will select and watch your video content. Carousels display a group of videos or playlists that you have defined in Video Cloud Studio. Using the Brightcove Beacon Administrative Console, you can add carousels to an existing page.

  • Store details: https://developer.roku.com/overview
  • Developer Account Setup: https://my.roku.com/signup
  • Suggested lead time for approval: Three weeks
Note that it can take up to 48 hours after publishing for your channel to be searchable on all devices. From the Roku site: My channel was just published, why can't I find it with Search?

Required information for Roku:

  • Poster and screenshot requirements
    • Channel Poster (540x405)
    • Screenshots up to 1920x1080
  • Developer Email
  • Preferred URL for more information
  • Administrative Contact (Name / Phone / Email)
  • Technical Contact (Name / Phone / Email)
  • Client to set up App logins for QA in store

Getting started

To add a carousel to an existing page, follow these steps:

  1. In a browser, navigate to the Roku Sign in page.

  2. Either login with your account or create a new account.

Adding a channel

Learn how to add a channel.

  1. From the top navigation, select the Dashboard link.

  2. From the dashboard, you can manage your channels. Select the Add Channel button.

  3. When the Developer SDK option is chosen, you will see options for a public or a non-certified channel.

  4. Select Public when you want the channel to go to the store. Select Non-certified when you want to create a testing channel.

    The channel name must be unique across both Public and Non-certified channels.

    We recommend the following:

    • Create your Public channel name, for example Destinations.

    • Create a Non-certified testing channel with a suffix of QA. So, in this case it would be named Destinations QA.

    First, we are creating the public channel.

  5. Next, we are creating the testing channel.

Managing properties

Properties are associated with each channel.

  1. Return to your Developer Dashboard and select the Manage My Channels link.

  2. For your channel, select the Preview and Update button.

  3. Expand the Preview and Publish dropdown menu, and select Properties.

  4. You should see a list of channel properties. On the left side, you will see a list of Channel Store countries where your channel will appear. For testing, you can select all countries. For your live channel, choose the countries where you want your channel to be seen.

  5. For accessibility features and compliance, select Yes. All Brightcove Beacon apps meet these requirements.

  6. Based on your content, choose the appropriate content rating.

  7. Select whether a customer account is required to access your channel features.

  8. Enter the URL for the privacy policy associated with your entire app.

  9. The Vanity Access Code is automatically generated by Roku. You can leave this value.

  10. Select the languages that you want to support.
