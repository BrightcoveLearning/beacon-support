Prerequisites

To be able to install the Brightcove Beacon app in your Samsung Smart TV, you will need:

  • Samsung Smart TV with the Tizen Operating System
  • Make sure that the Samsung Smart TV has the latest firmware installed

Installing the Brightcove Beacon App in Samsung Smart TV devices

Once you receive the Brightcove email, please perform the following steps:

  1. In the email sent by Brightcove, a ZIP attachment will come. Download the ZIP file.
  2. Double click the ZIP file and extract the folder from the file. The Brightcove Beacon installment files are on that folder.
  3. Copy the folder and paste it into the root of a USB flash drive.
  4. Insert the USB flash drive into your Samsung Smart TV. The Brightcove Beacon App will start to install automatically.
  5. In the Samsung Smart TV go to the Apps menu and open the Brightcove Beacon App.
Every time you want to install a new version of the Brightcove Beacon App, you will need to delete the previously installed version, otherwise you will find a system error.
