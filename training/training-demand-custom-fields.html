Training on Demand: Using Video Cloud Videos/Custom Fields

Viewing the Using Video Cloud Videos/Custom Fields training course is a great way learn how to setup the connection between Video Cloud and Brightcove Beacon. Part of implementing this connection is using metadata on the Video Cloud videos so when they are ingested into Brightcove Beacon they function as intended. This metadata is stored in custom fields, so you will need to understand how to create and populate Video Cloud custom fields.

There is a logical order to the videos in this course. So it is best to watch the videos in order of their appearance below.
