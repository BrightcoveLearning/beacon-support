Ad configurations define various aspects of ad playback, including ad calls, timing, beacons and other ad configuration options. The Server-Side Ad Settings define the ads that can be requested during live events. Server-side ads take advantage of Brightcove's Server-Side Ad Insertion (SSAI) technology and provide these main features:

  • Since ads are stitched into the live video stream on the server side, they are not affected by ad blockers
  • Ads are inserted at cue points sent from your encoder or you can create an instant cue point using the Live page Control Room
  • You can configure fill slates to fill any unused ad time

To learn more about server-side advertising, see Implementing Server-Side Ads with the Brightcove Player.

Note:
  • Before you can use SSAI, your Brightcove Beacon account needs to be configured for Dynamic Delivery and enabled for SSAI. Contact your account manager for details.
  • VMAP for Brightcove Live SSAI is not currently supported.
  • Although there is no prefetch functionality for SSAI ads, we delegate the ad position logic to the ad provider. Dynamic Delivery makes all the add calls up front. If we get an ad from the ad provider and it is transcoded in our system we will insert it into the stream.

The Server-Side Ad Settings page is used to manage the ad configurations. Note that to access the Server-Side Ad Settings page, you must be a user with the Administrator role.

To access the Server-Side Ad Settings page:

  1. In the left navigation, click Admin. Note that you must be an account administrator to access to the Administration page.
  2. Click Server-Side Ad Settings. A list of all ad configurations will be displayed.

Creating an ad configuration

To create an ad configuration, follow these steps.

  1. On the Server-Side Ad Settings page, click Create Ad Configuration.
  2. Enter a Name.
  3. Select the Ad Server Response:
    • VAST
    • DFP (Ad Rules)
    • Smart XML
  4. Enter the Ad Tag.
  5. Click Save.

To edit an ad configuration, click the ad name, make the changes and then click Save. To delete an ad config, check the box next to the ad and then click Delete. To duplicate an ad config, check the box next to the ad and then click Duplicate.

Once Live ad configurations have been created, when an event is created with the Enable Server-Side Ad Insertion (SSAI) property enabled, an ad configuration can be selected in the Control Room. For information on creating live events, see Creating and Managing Live Events using the Live Module.
