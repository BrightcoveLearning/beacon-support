Overview

To create an advertisement configuration you will need to do the following:

  • Obtain an ad tag server URL from your ad service provider. This will be used in Video Cloud Studio to create your SSAI ad configuration.
  • In Video Cloud Studio, create your SSAI ad configuration. At completion of the task you will copy the ad server ID to create your advertisement configuration in Brightcove Beacon Classic.
  • In Brightcove Beacon Classic create your advertisement configuration, which you will then be able to apply to your video content.

Creating an SSAI ad configuration ID

Follow these steps to create an SSAI ad configuration:

  1. In Video Cloud Studio, navigate to Admin â†’ Server-Side Ad Settings.
  2. From the Create Ad Configuration dropdown, select VOD
  3. Add a name for the ad config.
  4. Select your ad response - VMAP or VAST is recommended. If you use VAST, you will need to add cuepoints to the videos to specify where ad breaks should appear.
  5. Paste in your ad tag.
  6. Click Save.
  7. Find your new ad configuration in the table and copy your ad config ID. You will need this value for the Beacon advertising configuration.

Advertisement

To create an advertisement, follow these steps:

  1. Login to Brightcove Beacon.
  2. In the top navigation, select Advertisement .
  3. Select the Add New Configuration button.
  4. Here there is only one tab, that being for Basic Data .

  5. Enter values for the following fields:

    Field Required Notes
    Name Yes A descriptive name for your advertisement definition. This could be your campaign name.
    Provider Yes The value is Brightcove SSAI. This allows you to stitch ads into your stream using Brightcove Server-Side Ad Insertion (SSAI) .
    Live Ad Config ID No The ID for your live ad configuration created in Video Cloud. Either this or the VOD Ad Config ID is required.
    VOD Ad Config ID No The ID for your VOD ad configuration created in Video Cloud. Either this or the live Ad Config ID is required.

    You will create the ad configuration in Video Cloud Studio. For details, see the Video Cloud Ad Configuration ID document.
  6. Select the Create New Advertisement button.
